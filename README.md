10-25 POWERSHELL COMMANDS:Get-Location - shows current folder
Get-ChildItem - lists files and folders
Clear-Host - clears screen
Get-Date - shows date and time
Get-Process - shows running processes
Get-Service - shows services
Get-Command - lists all commands
Get-NetIPAddress - shows IP address
Test-Connection google.com - tests internet connection
New-Item test.txt - creates a file
New-Item test -ItemType Directory - creates a folder
Remove-Item test.txt - deletes a file
Remove-Item test -Recurse - deletes a folder
Copy-Item test.txt copy.txt - copies a file
Move-Item test.txt Desktop - moves a file
Rename-Item test.txt new.txt - renames a file
Set-Location Desktop - opens Desktop folder
Get-Content file.txt - reads file content
Set-Content file.txt "hello" - writes to a file
Stop-Process -Name notepad - closes Notepad
Restart-Computer - restarts computer
Stop-Computer - shuts down computer
Start-Service wuauserv - starts a service
Stop-Service wuauserv - stops a service
Get-Help Get-Process - shows help for a command
